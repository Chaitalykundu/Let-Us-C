# Q: What is the hexadecimal equivalent of the following binary numbers

&nbsp;

>
    1. 01011010
    2. 11000011
    3. 1010101001110101
    4. 1111000001011010

&nbsp;

## ANS

&nbsp;

```md
    1.  0101, 1010

    0101 = 5
    1010 = 10 = A

    0101, 1010 = 5A
```

&nbsp;

```md

    2. 1100, 0011

    1100 = 12 = C
    0011 = 3

    1100, 0011 = C3
```

&nbsp;

```md

    3. 1010, 1010, 0111, 0101

    1010 = 10 = A
    0111 = 7
    0101 = 5

    1010, 1010, 0111, 0101 = AA75
```

&nbsp;

```md

    4. 1111, 0000, 0101, 1010

    1111 = 15 = F
    0000 = 0
    0101 = 5
    1010 = 10 = A

    1111, 0000, 0101, 1010 = F05A
```
